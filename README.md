# 🧮 Smart Math Solver

A modern **Mathematics Web Application** built using **HTML, CSS, and JavaScript**.
This project allows users to perform multiple mathematical operations instantly with a clean UI and smooth animations.

---

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🔢 Power Calculation
* √ Square Root
* 📊 Percentage Calculator
* 🎨 Smooth Hover Effects
* ⚡ Interactive UI

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the web page
* **CSS3** – Styling and animations
* **JavaScript** – Logic and calculations

---

## 📂 Project Structure

```
Smart-Math-Solver
│
├── index.html
├── style.css
└── script.js
```

---

## ▶️ How to Run the Project

1. Download or clone the repository
2. Open the folder
3. Run **index.html** in your browser
